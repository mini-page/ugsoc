# UG-SOC Portfolio Project Context

## Overview
This project is a **Portfolio Enhancement Package** designed to transform a standard portfolio into a high-performance, modern web application.

## Key Features
- **PWA Support**: Manifest and Service Worker (`sw.js`).
- **Visual Effects**: Matrix Rain, Typing Animation, Parallax (`enhancements.js`).
- **Performance**: Lazy loading, caching.
- **UI Components**: Loading screen, scroll progress, resume downloader.

## Structure
- `index.html`: Main entry point.
- `enhancements.js`: Core logic for animations and interactivity.
- `sw.js`: Service Worker for offline capabilities.
- `manifest.json`: Web App Manifest.
- `components-snippet.html`: HTML snippets for components.
- `style.css`: Custom styles.

## Configuration
- Service Worker caching is configured in `sw.js`.
- Animation settings are in `enhancements.js`.
- Theme colors are managed via Tailwind CSS classes.

## Development
- Uses **Tailwind CSS**.
- **FontAwesome** for icons.
